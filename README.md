# canopus-jd-1-controller-mapper-windows
This is a software driver to make the old Canopus JD-1 video deck controller work with windows 11

<img width="1600" height="1600" alt="image" src="https://github.com/user-attachments/assets/e29d1c1f-1218-4cc5-9310-e828a70974c2" />

Just open and run the exe file and this window will show letting you map the keys to any keyboard keys you want. 

<img width="441" height="596" alt="image" src="https://github.com/user-attachments/assets/f6ac9f13-0524-4090-a64a-d29f39919b65" />
